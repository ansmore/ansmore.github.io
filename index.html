Web de Afiliados

Proyecto ERP Escolar

Desarrollo de un ERP para un proyecto escolar, utilizando Spring Boot y tecnologías modernas como Hibernate, JPA, y Thymeleaf. Aplicación de Presentaciones

Proyecto escolar enfocado en el desarrollo de una aplicación web para crear presentaciones dinámicas. Utilizando tecnologías como PHP, JavaScript, y CSS, el equipo logró implementar una solución que permite a los usuarios generar presentaciones con diferentes tipos de diapositivas. Este proyecto destaca por su enfoque modular y la implementación de funcionalidades avanzadas como la personalización de temas y la gestión de contenido multimedia.

El Punt Lila

El Punt Lila es una iniciativa innovadora destinada a promover la concienciación y proporcionar asesoramiento sobre agresiones sexuales en entornos de ocio nocturno. Desarrollado como un proyecto escolar, esta aplicación móvil facilita un punto de contacto directo para víctimas de agresiones Implementado con tecnologías ágiles como Scrum y Kanban, y utilizando herramientas como Trello y GIT para la gestión de tareas y el control de versiones, respectivamente. La aplicación destaca por su accesibilidad e interfaz intuitiva.

EventoPass

EventoPass es un portal web innovador diseñado para la venta y gestión de entradas a eventos variados, facilitando una plataforma centralizada para promotores. Este proyecto, destaca por su modelo de negocio basado en comisiones sobre ventas, una interfaz intuitiva para usuarios y promotores, y un sistema de validación de entradas eficaz. Implementado con tecnologías avanzadas, EventoPass representa una solución completa para la dinamización del sector de eventos.

API de EventoPass

La API de EventoPass es una solución integral para la gestión y optimización de imágenes, diseñada para enriquecer la experiencia de usuario en el portal de EventoPass. Ofrece funcionalidades como carga y optimización de imágenes, accesible a través de una interfaz RESTful. Esta API refuerza el portal EventoPass, permitiendo una gestión eficiente de recursos visuales en múltiples formatos y tamaños, adaptándose a las necesidades de promotores y usuarios finales en un entorno dinámico y visualmente atractivo.

Aplicación Móvil de EventoPass

La aplicación móvil de EventoPass, desarrollada con Cordova, es la herramienta esencial para la validación de tickets en eventos. Facilita a los organizadores la gestión de entradas a través de un proceso de validación rápido y seguro, mejorando la experiencia tanto de organizadores como de asistentes. La app permite escanear códigos QR de los tickets, verificar su autenticidad en tiempo real, y gestionar el acceso a los eventos de manera eficiente. Diseñada para ser intuitiva y fácil de usar, esta aplicación se integra perfectamente con el ecosistema de EventoPass, asegurando un manejo óptimo de la asistencia y contribuyendo a la realización de eventos exitosos.
